# VSCode

- Install extesions `vscode/extensions.sh`
- To read the configs from current VSCode installation run `vscode/configs_read.sh`
  - Creates `old_global_settings.json` in `$HOME/Library/Application\ Support/Code/User/`
- To write the configs from this directory to VSCode configs run `vscode/configs_write.sh`
  - Creates `old_settings.json` in this directory
